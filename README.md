# Game Log Tracker

Повноцінний Full-Stack застосунок для відстеження пройдених ігор та відкритих кінцівок. Дозволяє вести власну бібліотеку ігор та контролювати прогрес їх проходження.

##  Використані технології
- **Backend:** NestJS, TypeORM, PostgreSQL
- **Frontend:** React, Vite, CSS
- **Архітектура:** REST API, CRUD операції, зв'язки One-to-Many

##  Інструкція з встановлення та запуску

### 1. Підготовка бази даних
Переконайтеся, що у вас встановлений PostgreSQL. Створіть нову базу даних із назвою `game_log_db`.

### 2. Запуск Backend-частини
Відкрийте термінал у папці `backend` та виконайте команди:
npm install
npm run start:dev

Бекенд запуститься на `http://localhost:3000`.

### 3. Запуск Frontend-частини
Відкрийте новий термінал у папці `frontend` та виконайте команди:
npm install
npm run dev

Перейдіть за посиланням, яке видасть термінал (зазвичай `http://localhost:5173`), щоб відкрити застосунок у браузері.і
